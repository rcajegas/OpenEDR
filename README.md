
How to run this Free EDR: First clone or download the repository and install the dependencies using pip install -r requirements.txt. Then start the EDR server by running python server.py, which launches a Flask service on http://localhost:5000 that receives and stores alerts from monitored machines. In a second terminal window, run python agent.py, which acts as the endpoint sensor that continuously checks running processes on the system and sends an alert to the server if it detects suspicious commands like encoded PowerShell or tools such as mimikatz. Finally, open index.html in your browser and click Refresh Alerts to view any detections being reported by the agent in the dashboard.
